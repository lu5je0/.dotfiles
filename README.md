# Dotfiles

Personal infrastructure for a consistent, reproducible, and high-efficiency development environment across macOS, Linux, and WSL.

## Why This Repo

- Unified shell, editor, terminal, and automation setup
- Fast bootstrap for a clean machine
- Modular structure for continuous iteration

## Quick Start

```bash
git clone https://github.com/lu5je0/.dotfiles.git ~/.dotfiles
bash ~/.dotfiles/scripts/setup.sh
```

## Included Components

- Shell: `zsh`, aliases, prompt, workflows
- Editor: `neovim`
- Automation: scripts for setup, restore, and environment helpers
