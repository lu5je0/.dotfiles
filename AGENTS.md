# Dotfiles

个人 dotfiles 仓库，管理 shell、编辑器、终端与自动化环境配置。

## 目录结构
- `vim/`: Neovim 配置（有独立 `AGENTS.md`）
- `zsh/`, `zshrc`, `bashrc`: Shell 配置
- `scripts/`: 安装与环境初始化脚本（`setup.d/` 有独立 `AGENTS.md`）
- `skills/`: 本地 skills，清单见 `skills/INDEX.md`
- `submodule/`: 个人工具项目

## 目录协作
- 改子目录内容时，先读该目录下的 `AGENTS.md`（如果存在）。
